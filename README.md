# Py Map Stitcher QGIS Plugin

## Description
Py Map Stitcher is a QGIS plugin for downloading and stitching XYZ map tiles into georeferenced TIFF mosaics.

Features:
- Google Hybrid preview
- XYZ tile download
- TIFF export
- TFW + PRJ georeferencing, same method as the old working standalone script
- QGIS map extent selection
- Resume/cache support

## Installation
1. Open QGIS
2. Plugins → Manage and Install Plugins
3. Install from ZIP
4. Select the plugin ZIP

# PyMapStitcher 3 Standalone:

PyMapStitcher 3 is a desktop application for downloading, stitching, and exporting very large satellite map areas as GeoTIFF/BigTIFF files. 
The software supports GPU acceleration with NVIDIA CUDA and CuPy, direct GeoTIFF georeferencing, WebView-based map selection,
and high-performance tile processing for large-scale mapping workflows. 
https://doi.org/10.5281/zenodo.20480746
